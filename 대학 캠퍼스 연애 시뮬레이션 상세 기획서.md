# [개발용 상세 기획서] 대학 캠퍼스 연애 시뮬레이션 시스템

## 1. 게임 개요 및 구조
*   **배경:** 대학 캠퍼스에서의 한 학기.
*   **장르:** 선택, 육성, 아이템 사용이 결합된 연애 어드벤처 시뮬레이션.
*   **진행 흐름:** 개강 → 1차 이벤트 → 단체 이벤트(축제) → 2차 이벤트 → 단체 이벤트(MT) → 3차 이벤트 → 고백 이벤트 → 엔딩.
*   **시간 시스템:** 주요 이벤트일을 제외하고 매일 **낮(공강)과 밤, 총 2회의 자유행동**을 수행하며, 그 사이 시간은 내러티브 요약으로 처리함.

## 2. 플레이어 스탯 및 자유행동 로직
### 2.1. 주요 스탯
*   **체력, 지성, 사교성, 끈기, 피로, 자산(돈)**.

### 2.2. 자유행동 세부 수치
| 행동 | 효과 | 피로 변화 | 설명 |
| :--- | :--- | :--- | :--- |
| **운동** | 체력 +3 | +0 | 아이템(프로틴)과 조합 가능 |
| **공부** | 지성 +3 | +5 | 피로도가 높음. 비타민/아르기닌과 궁합 |
| **알바(편의점)** | 끈기 +1, +20,000원 | +5 | 기본 노동, 끈기 소량 상승 |
| **알바(상하차)** | 끈기 +2, +50,000원 | +15 | **하루 1회 제한**. 끈기 크게 상승 |
| **투자** | 랜덤 ±50~100% | +0 | **자산 30,000원 이상** 시 가능 (스탯 직접 변화 없음) |

## 3. 히로인 공략 및 포인트 시스템
### 3.1. 히로인별 공략 데이터
| 이름 | 선호 스탯 | 임계치 | 난이도 | 공략 조건 요약 |
| :--- | :--- | :--- | :--- | :--- |
| **하예은** | 체력 | 32 | 쉬움 | 이벤트 꾸준히 선택 + 대화 일부 성공 |
| **서다은** | 지성 | 35 | 보통 | 이벤트 + 대화 6~7회 + 3차 선물 필요 |
| **이봄** | 사교성 | 39 | 조금 어려움 | 대화 대부분 성공 + 선물 1~2회 필요 |
| **도희원** | 끈기 | 43 | 최고 난이도 | 대화 전부 성공 + 선물 2회 + 미니게임 성과 |
| **로아** | 특수(피로) | 46 | 히든 | **모든 이벤트 로아 선택** + **고백 시 피로도 70 이상** |

### 3.2. 포인트 구성 및 산정 규칙
*   **이벤트 (60%):** 총 27점 (1차:+3, 축제:+4, 2차:+6, MT:+5, 3차:+9).
*   **대화 선택 (20%):** 총 15회, 성공 시 회당 +1점 (감점 없음).
    *   세부: 1차(+2), 2차(+2), 3차(+3), 메신저(+3), 일상(+5).
*   **선물 (15%):** 2차/3차 이벤트에서만 반영. 최대 +8점.
*   **미니게임 (5%):** 1차(최대 +2), 2차(최대 +3) → 전체 합계 최대 +4점 (A~S랭크 달성 시).
*   **스탯 보너스:** 고백 시 해당 히로인 이벤트 1회 이상 참여 및 선호 스탯이 **1위면 +3점, 공동 1위면 +1점**.
*   **선택 복구:** 3차 데이트에서 다시 이전 히로인에게 복귀 시 **+2점 보정**.

## 4. 아이템 시스템 및 상점
### 4.1. 사용 규칙
*   **시점:** 자유행동 전/후에만 사용 가능 (행동 도중 불가).
*   **제한:** 1회 행동당 1개. **동일 날짜에 같은 아이템/태그 사용 시 2회차부터 효과 50% 적용**.
*   **효과:** "세션" 표기 아이템은 자유행동 1회 동안만 일시적 보정.

### 4.2. 선물 아이템 계층 규칙
가격과 관계없이 계층에 따라 포인트가 결정됨 (가격은 현실감 및 자원 관리용).

*   **중급(1만~3만대):** 2차(+2), 3차(+3)
*   **고급(4만~7만대):** 2차(+3), 3차(+4)
*   **최고급(≥8만):** 2차(+3), 3차(+5)

## 5. 특수 조건 및 엔딩
*   **로아(Hidden) 전용 규칙:**
    *   고백 이벤트 진입 시 **피로도 ≥ 70**일 때만 고백 버튼 활성화 (아니면 흑백 처리).
    *   피로도 수치에 따른 보너스: 70~79(+3), 80~89(+6), 90~100(+10).
    *   스탯 보너스(+3/+1) 규칙은 적용되지 않음.
*   **엔딩 구성:** 히로인별 해피/새드 엔딩, 로아 메리배드엔딩, 솔로 엔딩 등 **총 10종**.
*   **기타:** 해피엔딩에만 전용 CG가 존재함.
