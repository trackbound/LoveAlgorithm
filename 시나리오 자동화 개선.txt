로아 등장 시 오버레이 스크립트 작성법과 시나리오 문서 업데이트 관련해서 안내해 드립니다.

로아 등장 및 퇴장 시 아래와 같은 규칙으로 스크립트를 작성해 주시면 됩니다. 글리치 효과음은 별도 표기없어도 자동 재생됩니다.

Char,,C:Enter:Roa:Default:Roa_Mob_Default
로아가 '기본 표정' + '기본 모바일 오버레이'로 등장합니다.

Char,,C:Enter:Roa:Glare:Roa_PC_Negative
로아가 '째려보는 표정' + '화난 PC 오버레이'로 등장합니다.

Char,,C:Exit
퇴장 대상이 로아일 경우, 오버레이도 자동으로 함께 사라집니다.


현재 구글 드라이브에 가독성, 편의성, 자동화를 크게 개선한 시나리오(업데이트) 문서를 새로 업로드해 두었습니다.
 
원래 데모 버전까지는 기존 문서로 작업을 진행하고, 제가 스크립트를 2차로 가공해서 게임에 올리고 있었는데요! 
다음과 같은 문제들이 우려되어 작업 방식을 지금 동기화하고자 합니다.
- 제가 2차 가공을 거치면서 작업 속도가 지연되는 문제
- 갱햄님이 고민해서 작성해 주신 원본 느낌이 게임에 100% 빠르고 정확하게 반영되지 않을 수 있다는 점

새로 업데이트된 문서에서 바로 작업을 진행해 주시면 게임에 훨씬 매끄럽게 적용될 수 있을 것 같습니다!!

추가로 </>형태의 인라인 연출은 Text타입에서만 사용 가능합니다!!

추가 설명


1. 
Next 컬럼은 Char, BG 등 몇개 타입을 제외하고는 비워두어도 연출 흐름이 잘 돌아가도록 하였습니다.
시나리오 직접 보시면 더 잘 이해되실 겁니다!

2. 
- 기존에 간접적으로 위치 표기를 부탁드렸던 항목(자동저장)
- 시스템적인 부분
- 새 기능들 (SD, Place 등)
등이 시나리오에 작성되어 있습니다.


* 자동저장 
Flow,,Save - 자동저장만
Flow,,LoadingScene - 자동저장+로딩씬

* 좌상단 알림
Place,,[1일차 아침] | 침대 위

* CG, SD

* FX,,Wait (단순대기 버퍼)

* 흔들기 효과
- FX,,DialogueShake
- FX,,CamShake
- FX,,CharShake
세가지 흔들기 모두 Weak, Medium, Strong으로 조절 가능하며 생략시 Medium으로 작동합니다.
(ex- FX,,CamShake:Medium)

* 씬 정리, 시작 효과
- FX,,SceneEnd - 암전 후 일괄정리

- FX,,SceneStart:배경:EyeClose - '배경'은 세팅되지만 눈 감은 상태. Text 출력시 검은배경+대사창
// 이후 FX,,EyeOpen을 사용해야 SceneStart에서 세팅한 배경(자취방_침대위_낮)이 보임.

- FX,,SceneStart:배경 - '배경'에서 페이드인되며 바로 시작
// 671줄의 3일차(프롤로그) 시작 시, 침대가 아닌 학생회관 게시판에서 바로 시작하므로 EyeClose가 없는 SceneStart 사용

SceneEnd,SceneStart는 하루의 끝, 시작 시점에 사용해주시면 됩니다.
로딩씬이 필요할경우 SceneEnd, SceneStart 사이에 작성해주시면 됩니다. (시나리오 참고, 다른 위치에도 사용가능)
단순 배경전환이나 페이드와는 다르게 내부적으로 씬을 정리하고 새로 세팅합니다.
