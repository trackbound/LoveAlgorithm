[프롬프트] 러브 알고리즘 UI 개발 명세서
프로젝트 개요: '러브 알고리즘'이라는 게임의 메인 화면 및 서브 UI 시스템을 구현합니다. 모든 서브 UI(불러오기, 엑스트라, 설정)는 메인 화면 위에 오버레이 형태로 출력되며, 우측에서 좌측으로 슬라이드되는 애니메이션을 가집니다.
1. 메인 화면 (Main Screen)
구성 요소:
• 게임 제목 및 로고:
    ◦ 언어 설정에 따라 한국어 또는 영어 로고를 출력합니다.
• 메뉴 버튼 (6개):
    ◦ START (게임 시작), CONTINUE (이어하기), LOAD (불러오기), EXTRA (엑스트라), CONFIG (설정), EXIT (종료).
    ◦ 인터랙션 로직: 기본적으로 영어 텍스트로 표시되나, 마우스 오버(Hover) 시 한국어로 텍스트가 변경되어야 합니다 (예: LOAD -> 불러오기).
    ◦ 기능:
        ▪ START: 게임 처음부터 시작.
        ▪ CONTINUE: 마지막 저장 시점부터 시작.
        ▪ LOAD/EXTRA/CONFIG: 해당 UI 슬라이드 출력.
        ▪ EXIT: 게임 종료.
• 하단 푸터:
    ◦ 저작권 문구: "Copyright 2024 Team LUCID all right reserved" (언어 무관 영어 출력).
2. 불러오기 화면 (Load UI)
동작 방식:
• 메인 화면 위에서 오른쪽에서 왼쪽으로 슬라이드하며 등장합니다.
• 상단 바(Top Bar)는 팀 로고, 시간, 시스템 아이콘(블루투스, 와이파이, 배터리)을 포함하며 언어 무관 동일 디자인입니다.
주요 기능:
• 제목 표시: 한국어 설정 시 '불러오기', 그 외 'LOAD' 텍스트 출력.
• 세이브 데이터 슬롯 (Grid):
    ◦ 한 페이지당 6개 슬롯 표시.
    ◦ 각 슬롯 구성: 썸네일 이미지, 챕터 이름, 저장 시간(YYYY/MM/DD).
• 페이지 네비게이션:
    ◦ 현재 페이지 / 전체 페이지 표시 (예: 1 / 10).
    ◦ 이전/다음 페이지 이동 버튼 (첫/마지막 페이지 예외 처리 필요).
• 기능 버튼:
    ◦ 뒤로 가기: UI 닫기 (이전 화면 복귀).
    ◦ 삭제 모드: 삭제 버튼 클릭 후 데이터를 선택하면 삭제 팝업 출력.
• 팝업 (Popup):
    ◦ 데이터 로드 시: "해당 데이터를 불러오시겠습니까?" (네/아니요).
    ◦ 데이터 삭제 시: "해당 데이터를 삭제 하시겠습니까?" (네/아니요).
    ◦ 팝업은 화면 중앙 정렬, 배경 클릭 시 팝업 닫기.
3. 엑스트라 화면 (Extra UI)
동작 방식:
• LOAD UI와 동일하게 우측에서 좌측으로 슬라이드 출력.
탭(Tab) 구성:
1. Scene: 씬 다시보기. 썸네일과 씬 이름 그리드 출력.
2. CG: 수집된 CG 이미지 보기. 썸네일과 CG 이름 그리드 출력.
3. 수집(대화): 수집 요소/대화 보기. 썸네일과 이름 그리드 출력.
공통 요소:
• 상단 바 및 네비게이션 바 (뒤로 가기, 닫기 버튼 포함).
4. 설정 화면 (Config UI)
동작 방식:
• 우측에서 좌측으로 슬라이드 출력.
• '설정 초기화' 버튼 포함.
설정 항목:
1. 화면 설정:
    ◦ 모드: 전체화면 / 창모드 (선택 시 하이라이트, 둘 중 하나 필수 선택).
    ◦ 해상도: 좌우 버튼으로 선택 후 '적용' 버튼 클릭 (1920x1080, 1600x900 등 목록 제공).
2. 음량 설정 (Slider):
    ◦ 배경음, 효과음 볼륨 조절.
    ◦ 캐릭터별 음성: 하예은, 서다은, 이봄, 도희원, 신유진, AI 로아 각각 개별 볼륨 슬라이더 제공.
3. 속도 설정 (Slider):
    ◦ 대사 출력 속도, 자동 진행 속도 조절.
5. 데이터 구조 (Localization)
• 다국어 지원: 모든 텍스트는 ID 기반으로 관리됩니다.
• 테이블 구조: 식별번호(ID), 국가명(Nation), 국가코드(CountryCode).
• ID 예시: 1001~1999.
• 언어 설정 변경 시 해당 ID에 매핑된 텍스트로 UI가 갱신되어야 합니다.
