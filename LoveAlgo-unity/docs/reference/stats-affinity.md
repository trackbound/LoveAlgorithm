# 스탯 & 호감도 참조

## 플레이어 스탯

| 코드 | 한글 | 증가 수단 |
|------|------|----------|
| `Str` | 체력 | 운동, 상하차 |
| `Int` | 지성 | 공부 |
| `Soc` | 사교성 | 스터디그룹 등 |
| `Per` | 끈기 | 운동 |
| `Fatigue` | 피로 | 행동 시 누적, 휴식/아이템으로 감소 |

## 히로인 선호 스탯

| 히로인 | 선호 | 난이도 | 포인트 만점 |
|--------|------|--------|------------|
| 하예은 | Str | ★☆☆ | 300 |
| 서다은 | Int | ★★☆ | 300 |
| 이봄 | Soc | ★★★ | 300 |
| 도희원 | Per | ★★★★ | 300 |
| 로아 | Fatigue≥70 | 히든 | 300 |

## 호감도 포인트 구성

| 카테고리 | 비율 | 소스 |
|---------|------|------|
| 이벤트 선택 | 60% | 스토리 이벤트 선택지 |
| 대화 | 20% | 일반 대화 선택지 |
| 선물 | 15% | ShopManager.GiveGift() |
| 미니게임 | 5% | MiniGameLauncher (히로인당 최대 5점) |

## 스탯 보너스 (엔딩 판정 시)

| 조건 | 보너스 |
|------|--------|
| 히로인 선호 스탯이 플레이어 스탯 중 1위 | +3 |
| 히로인 선호 스탯이 공동 1위 | +1 |

## 주요 수치

| 항목 | 값 |
|------|-----|
| 하루 자유행동 횟수 | 2회 (오전/오후) |
| 피로 최대치 | 100 |
| 로아 히든 루트 피로 조건 | ≥70 |
| 아이템 중복 사용 감소율 | 50% |
| 투자 최소 금액 | ₩30,000 |
| 선물 히로인당 최대 포인트 | 8점 |
| 미니게임 히로인당 최대 포인트 | 5점 |
